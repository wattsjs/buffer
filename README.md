# Buffer

A native macOS IPTV player built with SwiftUI and libmpv.

## Features

- M3U playlist and Xtream Codes server support
- Electronic Program Guide (EPG) with search
- Live sports via ESPN integration
- Favorites, recents, and program reminders
- Catchup/rewind playback
- Multi-view player windows
- Channel group management (hide, reorder)
- Configurable stream buffering
- External player support
- Keyboard shortcuts
